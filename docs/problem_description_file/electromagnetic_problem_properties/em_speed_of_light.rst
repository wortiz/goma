******************
EM Speed Of Light
******************

::

	EM Speed of Light = <float> 

-----------------------
**Description / Usage**
-----------------------

This sets the speed of light :math:`c_0` for Time-Harmonic Maxwell problems

<float>                       
    Value of speed of light

------------
**Examples**
------------

Following is a sample card:
::

	EM Speed Of Light = 3e8
